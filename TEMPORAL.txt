Instalar Python en Windows
1 Descargar e intalar python en windows en una ruta de facil acceso
2 Agregar la ruta de instalcion a las variables de entorno de windows
3 Instalar virtualenv: pip install virtualenv
4 crear un entorno virtual : virtualenv nombre_carpeta
5 Activar el virtualenv dentro de la carpeta donde se creo el entorno virtual: Scripts\activate.bat
6 con el entorno activado se procede a instalar Django: pip install django
